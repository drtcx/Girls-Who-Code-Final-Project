Reporting Police Misconduct?

If you are a witness/victim of police misconduct you should talk with a lawyer about your option if you chose to pursue that route.

Find a location to file your compliant

Depending on what state you are in. It may be required to file a police report at different locations, some are in police stations, some have to be filed in offices. To find where to file in your state, you can google it. If you have to file in a police station, don't tell the officer(s) on duty too much about the situation.

File & Complete your Compliant Report

File out the report as accurate as possible, make sure to write out the event while it is fresh on your mind and you don't forget any details. make sure you include every detail while filing out the report. Once you are finish make a copy of the report and keep it inside a secure place in your house.

Send in your Compliant Report

When sending in your report it is best to send it in by mail and also send some additional copies to organizations that fight aganist police misconduct also advocate for victims of such crimes. For a in-depth explaination on police brutality visit The Police Brutality Attorney
